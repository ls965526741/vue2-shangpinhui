# 仿尚品汇项目

### 安装项目依赖

```
npm install
```

### 运行项目

```
npm run serve
```

### 选用技术

```js
vue
axios
nprogress // 加载进度条
swiper // 轮播图插件 注意版本问题
vue - lazyload // 图形懒加载
mockjs // 模拟前后端交互
```
